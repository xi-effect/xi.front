## Frontend репозиторий платформы Ξffect  

-------
### Для начала работы:

1. Склонируйте репозиторий

    `git clone https://github.com/xi-effect/xieffect-front.git`

2. Установите зависимости

    `npm i`

3. Запустите локальный сервер

    `npm run dev`
